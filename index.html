Well, before you start making rainbow text, we need to make classes

To make a class, you need to add class="class-name" before the > in a tag, example: <p class="tutorial-paragraph">Text

Well, what now? We use CSS, create a new file called Something.css and in the head section of your html add <link rel="stylesheet" type="text/css" href="SameSomthing.css">

In our new css file add:
.classname {
color: red;
}

Now all tags with that class will be red like this

There are multiple colours you can just write like color: yellow; or color: green; but if you want to get a certain colour you can use color: rgb(255, 0, 255);

rgb stands for red, blue and green, the numbers (0-255) is how much of that colour there is, so rgb(255, 0, 255) is 255 red, 0 green and 255 blue.

Huh? You want rainbow text, well, here is a way to do it.<\p>

<p> Normal text to <span class="red">r</span> <span class="orange">a</span> <span class="yellow">i</span> <span class="green">n</span> <span class="blue">b</span> <span class="indigo">o</span> <span class="violet">w</span></p>

Complicated, I know!

Make sure to add all of the classes to your css file!

Links

Using links you can send people to other websites, or even a shortcut to another part of the webpage.

To send someone to another website we need to <a href="link">Text<a>, example: <a href="https://finelime.github.io">Click Here</a>

But what if you want to open it in a new tab? it's simple, we use target="_blank", example: <a target="_blank" href="https://finelime.github.io">Click Here</a>


To send someone to another part of the same webpage we can use id's! instead of a link we use #idname, example: <a href="#Title">Click Here</a>

This is useful if you have a huge webpage.
